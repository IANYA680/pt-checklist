⚙️ Google API 설정 방법

구글 캘린더 연동을 위해 아래 단계를 따라 Client ID를 발급받으세요:

  1. console.cloud.google.com 접속 → 새 프로젝트 생성
  2. API 및 서비스 → 라이브러리 → "Google Calendar API" 활성화
  3. OAuth 동의 화면 설정 (외부, 테스트 모드)
  4. 사용자 인증 정보 → OAuth 2.0 클라이언트 ID 생성
  5. 승인된 JavaScript 원본에 현재 도메인 추가
